# git function 
git --version 版本
git config --global user.name " " 使用者名稱
git config --global user.email " " 使用者信箱 
git init 開啟
clear 清除
git status 查看檔案狀態

--git 檔案狀態
1. Untracked 未追蹤
2. Tracked 已追蹤
3. Staged 已暫存
4. Committed 已提交
Staging Area 暫存區

git add +檔案名稱(.md) 未追蹤 >已追蹤 顯示(A) 中間空白鍵相隔 檔案名一次加多個 
git commit -m " "
